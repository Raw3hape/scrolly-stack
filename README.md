# Scrollytelling Stack Demo

A premium 3D scrollytelling experience built with React, Vite, and Three.js.

## Getting Started

1.  **Install dependencies**:

    ```bash
    npm install
    ```

2.  **Start the development server**:

    ```bash
    npm run dev
    ```

3.  Open the local URL (usually `http://localhost:5173`) in your browser.

## Features

- **Responsive Layout**: Split screen on desktop, stacked on mobile.
- **3D Visualization**: Interactive 3D stack that synchronizes with scroll position.
- **Premium Aesthetics**: Glassy materials, smooth animations, and clean typography.
